## About This Repo

This repository provides **comprehensive Chinese/English tutorials and configuration guides** for Clash proxy tools.

## Quick Start

### Step 1: Get a Subscription

- [Clash for Windows](https://clashforwindows.site) - All-platform clients
- [Resource Hub](https://flclash.us) - Free rules and tools
- [Mobile Guides](https://clashmi.site) - Android/iOS setup

### Step 2: Download Clients

| Platform | Recommended Client | Link |
|----------|-------------------|------|
| Windows | Clash for Windows / Verge | [https://clashforwindows.site](https://clashforwindows.site) |
| macOS | ClashX / Clash Verge | [https://clashforwindows.site](https://clashforwindows.site) |
| Android | Clash for Android / v2rayNG | [https://clashmi.site](https://clashmi.site) |
| iOS | Shadowrocket / Stash | App Store |

### Step 3: Configure and Use

1. Open your client
2. Import subscription or add nodes manually
3. Select a node and enable proxy
4. Enjoy!
